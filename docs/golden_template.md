# Golden Template: 美股复盘

目标：每日自动生成一份与 Golden Sample《20260528美股复盘.pdf》高度一致的中文卖方风格美股复盘。报告不是数据审计表，也不是全量行情表，而是围绕“昨夜最大边际变化”和“AH盘前动作”的投研日报。

## 固定标题

```text
美股复盘 · YYYY年M月D日
```

顶部保留生成时间，PDF 中间显示主标题。

## 固定结构

### 一、美股指数概况

- 开头一段写昨夜美股整体判断。
- 指数表格只保留：指数 / 收盘 / 涨跌幅。
- 写一句话结论。
- 写 MA20 偏离。
- 回答三问：
  1. 风险偏好是全面回暖还是局部修复？
  2. 大盘 / 小盘谁更强？
  3. 科技 / 半导体是否仍是主导？

### 二、宏观与地缘

#### 2.1 美国经济及美联储降息

- 表格：事件 / 来源日期 / 要点。
- 写降息预期一句话判断。

#### 2.2 中国经济及政策

- 写昨夜重大增量，或明确“无重大增量”。
- 写对A股/港股影响。

#### 2.3 地缘冲突

- 表格：冲突 / 近2日状态 / 变化。
- 写对油气、黄金、航运、AH风险偏好的影响。

### 三、行业与ETF结构

#### 3.1 行业涨跌结构

使用项目符号写：

- AI软件应用
- 大型科技
- SaaS/IT服务
- 传统工业/能源

最后写“判断：……”。

#### 3.2 ETF结构

- 表格：ETF / 关注维度 / 信号。
- ETF 只保留 QQQ、SPY、SMH、SPY/QQQ量变等关键口径。
- 禁止把所有 ETF 的 MA5/MA20/RSI 大表格塞进正文。

### 四、美股科技股跟踪

#### 4.1 大型科技股（Magnificent 7）

- 表格：股票 / 涨跌幅 / 信号。
- 包含 MSFT、AMZN、NVDA、AAPL、GOOGL、META、TSLA。
- 表格后必须写结论。

#### 4.2 自选美股异动

- 只写当天真正有异动或强催化的股票。
- 没有公司级催化时，写“未匹配到明确公司级催化”，不要乱匹配。

#### 4.3 AI主线跟踪

- 开头一句写昨夜是否有新增 AI 关键业绩/指引。
- 回答三问：
  1. AI主线是在扩散、收敛，还是高位分歧？
  2. AI Capex、半导体、软件、端侧应用分别强弱如何？
  3. 业绩与指引是强化还是削弱 AI 主线？

#### 4.4 重要信息 / AI产业催化

- 表格：事件 / 来源日期 / 影响链条 / 对美股复盘结论 / 对A股盘前结论是否update。
- 只放真正高相关 AI 产业催化。

### 五、AH盘前参考

#### 5.1 美股映射

- 表格：美股信号/异动 / AH映射方向 / 盘前判断。
- 语言直接、可交易，例如：适合追强、看承接、不追高、等待确认。

#### 5.2 盘前参考

- 写一段总判断。
- 写 Update Matrix。
- 写最终判断，必须包含：
  1. AH偏多/偏谨慎/结构分化；
  2. 主线共振确认；
  3. 盘前最优先方向；
  4. 只看承接、不追高；
  5. 防兑现/风险提示。

## PDF 版式

- 目标 4 页左右，超过 6 页需要 warning。
- 一级标题红色，二级标题黑色加粗。
- 表格灰色表头、细边框。
- 不使用大面积密集行情表。
- 技术审计只进入 Telegram、日志或内部诊断，不进入正式正文。

## 正文禁止词

正式 Markdown/PDF 正文不得出现：

- HTTP
- Too Many Requests
- provider_chain
- 完整池行情可用
- 实时获取
- 缓存降级
- 失败 ticker
- attempt
- Python/HTTP 错误
- 大面积 N/A
